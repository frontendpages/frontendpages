## mental model (core idea)

- input data → vertex shader → interpolation → fragment shader → pixels

## how to approach

- start with fragment shaders → fastest visual feedback
- learn by drawing shapes, not reading theory
- treat shaders as **math + pixels**, not traditional programming
- experiment constantly
